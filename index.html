What is Machine Learning?

Machine Learning is a branch of artificial intelligence (AI) and computer science that deals with the creation of computer systems that learn from data, identify patterns and algorithms and make decisions with minimal human intervention.

Supervised Learning

Data scientists provide or specify both the input and output of the algorithm. These algorithms are labeled with training data and defined with variables for correlations.

Supervised learning algorithms are used in:

Binary & Multiclass classification, Ensembling and Regression modeling

@TechTarget

Unsupervised Learning

Don't require data to be labeled. They sift through unlabeled data to look for patterns that can be used to group data points into subsets. Most types of deep learning, including neural networks, are unsupervised algorithms.

Unsupervised learning algorithms are used in:

Clustering, Anomaly Detection, Association rule, Dimensional reduction

@TechTarget

Semi-supervised Learning

Works by using a small amount of labeled training data to an algorithm. The algorithm learns the dimensions of the data set and applies it to new unlabeled data. This allows a balance between the performance of supervised learning and the efficiency of unsupervised learning.

Semi-supervised learning algorithms are used in:

Machine Translation, Fraud Detection, Labeling Data

@TechTarget

Reinforcement Learning

Works by programming an algorithm with a distinct goal and a set of rules for accomplishing that goal. A data scientist will also program the algorithm to seek positive rewards for achieving its ultimate goal and to avoid punishments for moving farther away from its goal.

Reinforcement learning algorithms are used in:

Robotics, Video Gameplay, Resource Management
